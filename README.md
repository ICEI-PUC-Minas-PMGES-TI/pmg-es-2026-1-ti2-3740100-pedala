# PedaLa

O projeto PedaLá é uma plataforma web desenvolvida para facilitar a locação de bicicletas para pessoas físicas, atendendo a necessidades que vão desde passeios de lazer até o uso profissional, como em serviços de entrega. A iniciativa foi desenhada para contornar obstáculos comuns do dia a dia, como os altos custos de compra e manutenção de uma bicicleta própria, além da escassez atual de opções de aluguel que sejam simples e totalmente digitais. Com isso, o sistema busca incentivar a mobilidade urbana sustentável de maneira prática, acessível e mais organizada.

Para garantir uma experiência ágil e segura, o sistema permite que o cliente realize todo o processo online, o que engloba o cadastro, a reserva da bicicleta, o processamento do pagamento digital e a contratação de planos de seguro personalizados (que variam entre Básico, Intermediário ou Premium). O grande diferencial do projeto voltado à segurança é a implementação de tecnologia de rastreamento via GPS em todas as bicicletas. Essa funcionalidade garante maior proteção patrimonial e fornece um controle administrativo em tempo real, reduzindo ativamente os riscos de furto e assegurando o acompanhamento do equipamento antes, durante e após a locação.
